== Installation ==

<p>
Installation of <strong>RD Hover Effect</strong> is very simple. 

Option 1

1. Download the plugin rd-hover-effects.zip from download location.

2. Unpack the *.zip file and extract the RD Hover Effects folder.

3. Drop the RD Hover Effects plugin folder into your wp-content/plugins folder.

4. In word press administration panels, click on Plugins from the menu.

5. You should see your plug-in listed under Inactive plug-in tab.

6. To turn the word presses plug-in on, click activate.


Option 2

1. Download the plugin RD Hover Effects Plugin from download location.

2. Go to Add New menu under Plugins tab in your word-press admin.

3. Select upload link (top link menu).

4. Upload the available RD Hover Effects Plugin file and click install now

5. Finally click activate plug-in link to activate the plug-in.
